### What This Is
A single-file, fully self-contained HTML editorial that reframes the data-center conversation.  

It is **not** anti-tech. It is a clear, policy-sharp argument that communities asked to host the physical infrastructure of AI, cloud, and digital life deserve a real, enforceable deal — not just tax revenue and landscaping renderings.

Designed with glassmorphism, atmospheric orbs, and film-grain texture, it reads like a high-end print magazine but lives natively on the web.

### Key Sections
- Opening argument: Power, not just facilities  
- The numbers (Virginia GDP, polling collapse, PUE benchmarks)  
- The smarter deal — 6 concrete “runway” demands  
- Binding Community Benefit Agreements  
- Exit liability & decommissioning  
- Plain-language statement for neighbors  

### Tech
- Pure HTML + CSS (single file: `index.html`)  
- Glassmorphism + cinematic floating orbs  
- Responsive (mobile-first)  
- Print-friendly  
- Zero dependencies  

### How to Run Locally
1. Download `index.html`
2. Open it in any browser — nothing else required.
